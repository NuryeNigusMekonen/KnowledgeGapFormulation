# Week 12 Knowledge Gap Formulation

This directory contains the Day 1 submission package for the Week 12 Knowledge Gap Formulation workflow. The materials document the first daily cycle of question formulation, morning-call sharpening, explainer drafting, evening-call sign-off, and supporting source collection.

## Day 1 Scope

This Day 1 package is centered on one inference-time mechanics question grounded in prior portfolio work from TheConversionEngine. The question focuses on prefill, decode, KV or prefix caching, and cache invalidation, with direct relevance to prompt assembly, latency interpretation, and cost analysis in the Week 10 and Week 11 artifacts.

## Contents

- `question.md`: final sharpened question for the daily question vote
- `morning_call_summary.md`: summary of how the morning call clarified and narrowed the question
- `explainer.md`: full explainer addressing the selected knowledge gap
- `thread.md`: short public-thread version of the explainer
- `evening_call_summary.md`: summary of revision outcomes and final sign-off
- `sources.md`: canonical references and local project evidence used in the explainer

## Status

This folder is intended as the document set for Day 1 of Week 12. Later days in the week should continue as separate daily cycles with their own question, call summaries, explainer, thread, and supporting materials.
